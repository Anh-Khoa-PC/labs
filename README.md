# Nexus VN Security Research Lab

Chào mừng bạn đến với môi trường thử nghiệm bảo mật của Nexus VN.

### Cách sử dụng:
1. Nhấn vào nút **"Code"** -> **"Codespaces"**.
2. Chọn **"Create codespace on main"**.
3. Chờ môi trường khởi tạo để bắt đầu lab.

**Lưu ý:** Đây là một Proof of Concept (PoC) cho báo cáo bảo mật #3474011.
